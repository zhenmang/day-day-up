**算法复杂度**

当输入规模n足够大时，空间复杂度主要由临时存储变量所占空间决定，时间复杂度由基本操作的执行次数决定，而时间复杂度是空间复杂度的上限。故，如果将复杂度的问题极致的简化，就是看时间复杂度，将时间复杂度极致简化，就是与输入规模n的最高次幂之间的关系。

例如，以下算法的算法复杂度，可以认为是T(n) = O(n)

```
a=0;
b=1;                 ①
for (i=1;i<=n;i++)   ②
  {
     s=a+b;　　　　   ③
     b=a;　　　　　   ④  
     a=s;　　　　　   ⑤
  }
```

解： 语句1的频度：2,

 语句2的频度： n,

语句3的频度： n-1,

语句4的频度：n-1,

语句5的频度：n-1,

T(n)=2+n+3(n-1)=4n-1=O(n)
